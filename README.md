## Description
###### The HMS built using Python Django framework provides a user-friendly dashboard for both doctors and patients. The dashboard for doctors includes features such as appointment management, patient management, prescription management, and access to patient medical history. The doctors can also view and manage their schedules and patient appointments through the dashboard.
